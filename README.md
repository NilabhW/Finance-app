# Finance App

A personal finance and expense analytics app built with React.

## Features
- Add and manage income/expense transactions
- Search, filter, and sort transactions
- Monthly budget tracking with progress visualization
- Analytics dashboard with pie, line, and bar charts
- Currency conversion using live exchange rates
- Data persists across browser sessions via localStorage

## Tech stack
React · Vite · React Router · React Hook Form · Yup ·
Recharts · date-fns · Framer Motion · Axios

## Running locally
npm install
npm run dev
